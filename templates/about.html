What is PhishGuard?

PhishGuard is an intelligent email phishing detection tool designed to protect users from malicious emails. Leveraging the power of Natural Language Processing (NLP) and Machine Learning (ML), it analyzes various aspects of an email, including its content, subject, sender, and embedded URLs, to determine its legitimacy.

Our goal is to provide a robust and user-friendly solution that helps individuals and organizations identify and avoid phishing attempts, thereby enhancing their digital security posture.

Technologies Used
  • Backend: Python with Flask framework for the web application and API endpoints.
  • Machine Learning: Scikit-learn for building and training the phishing detection model.
  • Natural Language Processing: NLTK for text preprocessing, tokenization, and feature extraction.
  • Email Integration: Google Gmail API for scanning user inboxes (with user consent).
  • Frontend: HTML, CSS (Bootstrap 5 for styling), and JavaScript for interactive elements.
  • Deployment: Designed for deployment on platforms like Vercel.
